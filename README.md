# AI-Based-Pneumonia-Detection-from-Chest-X-Ray-Images-Using-CNN-Grad-CAM-with-Python-
Designed a deep learning model using CNN and Grad-CAM for automated chest X-ray analysis and pneumonia classification.

This project presents an AI-based pneumonia detection system that uses deep learning, specifically Convolutional Neural Networks (CNNs), to classify chest X-ray images as “Pneumonia” or “Normal.” Pneumonia is a serious respiratory disease that requires early and accurate diagnosis, yet manual interpretation of X-rays can be time-consuming and prone to human error, particularly in high-demand healthcare settings. To address this, the system automates detection by learning patterns from labeled medical images, using a publicly available chest X-ray dataset along with image preprocessing and data augmentation to improve performance. The model is implemented in Python within a cloud-based environment such as Google Colab, and its effectiveness is evaluated using metrics including accuracy, precision, recall, F1-score, and a confusion matrix. Additionally, Grad-CAM is employed to provide visual explanations of predictions, enhancing interpretability. Overall, the system functions as a decision-support tool for early pneumonia detection and highlights the potential of artificial intelligence in medical image analysis.
